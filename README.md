# Project End Agent

Task management agent that breaks down project goals into actionable sub-tasks, tracks progress, and generates completion reports.

## Features

- Goal decomposition into sub-tasks
- Progress tracking with status updates
- Dependency management between tasks
- Completion report generation

## Tech Stack

Python, LLM APIs

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Project-End-Agent.git
cd Project-End-Agent
pip install -r requirements.txt
```

### Usage

```bash
python agent.py
```
